Управление закупками товаров ()
Способность поиска, выбора и приобретения товаров и услуг, необходимых для ведения бизнеса. Включает в себя способности: Работа с поставщиками Управление категориями Обработка заказов на покупку Планирование заказов на покупку Оптимизация заказов на покупку Более развернуто: 1. **Supplier Agreement and Settlement** - Договорённости и расчёты с поставщиками. 2. **Purchase Order Processing** - Обработка заказов на закупку. 3. **Purchase Order Scheduling** - Планирование заказов на закупку. 4. **Invoice Management** - Управление счетами-фактурами. !!5. **Purchase Order Collaboration** - Совместная работа по заказам на закупку. 6. **Collective Purchase Order Management** - Централизованное управление заказами на закупку. 7. **Purchase Order Optimizing** - Оптимизация процесса заказов на закупку.
